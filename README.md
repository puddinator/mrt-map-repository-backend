# mrt-map-repository-backend
Backend for mrt map project

Website for online crowdsourced repository of train station layouts to help people with disabilities navigate unfamiliar stations.

Frontend built with React

Backend is a REST API built with Express.js, Mongodb database, and deployed on Heroku

Website: https://puddinator.github.io/mrt-map-repository/

Frontend: https://github.com/puddinator/mrt-map-repository
Backend: https://github.com/puddinator/mrt-map-repository-backend
